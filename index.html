Case Study - Image Sharing Platform

This was a case study to learn how to build and maintain an image hosting and sharing platform. Imagined as a cross between imgur and 9gag.

I originally started this project to learn more about handling file uploads to a server as well as to familiarize myself with the Python programming language and the Flask micro-framework.

The web application has gone through several iterations as I learned more about the problem domain. Some of the features that I've added over time:

  • User registration.
  • Email-based user authentication.
  • Role-based user management.
  • Ability for users to create posts.
  • Picture gallery management for user posts.
  • User commenting system.
  • Administrative functionality for authorized users.

AscendingWorld.com

Ascending World is a social media platform with a focus on increasing social awareness to bring about positive change.

I was asked to help implement back-end functionality for the web application. I lead design workshops and was an important part of the decision making process.

Most of my time spent with the Ascending World team involved setting up the database for the web application information, defining a normalized schema, and writing functions in PHP for CRUD interactions with the database. I promoted the use of the PDO module and bound parameters over the previous mysql implementation to increase site security.
